Unified Machine Learning Tool

Food Inspector

Food Inspector is an Android based application which has a Java Servlet based back end . The application is scans a fruit and lets the user know about the freshness of a fruit. The application is an online based application which sends the image to the Java Servlet back end running on our custom server which runs a tensor flow based Machine Learning model and returns the most likely result according to the model, upon the return of the result the Java Servlet also updates the Oracle Database saving the history and result of the scans for unique device IDs which is running on a separate server. The application is made on latest trend of minimalist UI and looks up to par with the latest trends.

Cute Algorithm (File Encryption Software)

Cute Algorithm is a JavaFX based file encryption software with our own developed symmetric encryption algorithm. The algorithm was named Cute Algorithm as it works on combination of simple manipulations which together makes the algorithm becomes very secure. The encryption algorithm is very efficient and very secure. The core encryption logic is unique and developed by our team independently. The core logic only takes help of a external Caesar Cipher to give the encryption an extra layer of security. The algorithm is able to encrypt files up to 4GBs successfully. The Software has a beautiful minimalist layout and is designed with latest trend in mind.
